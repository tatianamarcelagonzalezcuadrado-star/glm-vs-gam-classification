# TITTLE
# Comparison Between Logistic Regression and Generalized Additive Models for Binary Classification with Imbalanced Data and Nonlinear Predictors

## 📖 Project Description

This repository contains the code developed for my undergraduate thesis in Statistics at the University of Córdoba.

The research compares the predictive performance of **Logistic Regression (GLM)** and **Generalized Additive Models (GAM)** in binary classification problems under different levels of class imbalance and nonlinear relationships between predictors and the response variable.

The study combines **simulation experiments** with the analysis of a **real-world health dataset**, evaluating the models through cross-validation and several performance metrics, including the Area Under the ROC Curve (AUC), Accuracy, Precision, Recall, F1-score, Balanced Accuracy and Akaike Information Criterion (AIC).


## 🎯 Objectives

- Compare the predictive performance of Logistic Regression and Generalized Additive Models.
- Evaluate model behavior under different sample sizes and levels of class imbalance.
- Assess the effect of nonlinear predictors on binary classification.
- Apply both methodologies to simulated and real-world health data.
- Identify scenarios in which GAM provides advantages over traditional Logistic Regression.



## 📊 Methodology

The project was developed in two stages:

### 1. Simulation Study

Different simulation scenarios were generated by varying:

- Sample size
- Class imbalance level
- Correlation between predictors
- Nonlinear functional forms

For each scenario, Logistic Regression and GAM were fitted and evaluated using repeated simulations.

### 2. Real Data Analysis

Both models were applied to a real health dataset using variables such as:

- Age
- Body Mass Index (BMI)
- Health center (Site)

The response variable corresponds to disability status.

Model performance was evaluated using 5-fold Cross Validation.



## 📈 Evaluation Metrics

The models were compared using:

- Area Under the ROC Curve (AUC)
- Accuracy
- Precision
- Recall (Sensitivity)
- F1-score
- Balanced Accuracy
- Akaike Information Criterion (AIC)

## 📌 Main Findings

The results show that Generalized Additive Models (GAM) provide greater flexibility for modeling nonlinear relationships while maintaining predictive performance comparable to Logistic Regression. In simulation scenarios involving nonlinear effects, GAM generally achieved superior discrimination performance, whereas Logistic Regression remained competitive under approximately linear relationships.
